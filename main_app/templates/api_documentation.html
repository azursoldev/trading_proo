ðŸ’¡ Usage Examples

Start Scraping (JavaScript)

fetch('/api/start-scraping/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-CSRFToken': getCookie('csrftoken')
    },
    body: JSON.stringify({
        source: 'reuters',
        max_articles: 20
    })
})
.then(response => response.json())
.then(data => console.log(data));
